✊✋✌️ Rock Paper Scissors Game

A simple and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript. Play against the computer and track your score in real-time!

🚀 Features ------
🎮 Play Rock, Paper, Scissors against the computer
🤖 Random computer choice
📊 Score tracking (Win, Lose, Draw)
🔁 Reset score button
⚡ Auto play mode
🎨 Clean and modern UI design

🛠️ Technologies Used ---------
HTML – Structure of the game
CSS – Styling and layout
JavaScript – Game logic and interactivity

📸 Preview -------
![Screenshot](images/Screenshot_1.png)

🎯 How to Play -----------
Click one of the buttons:
✊ Rock
✋ Paper
✌️ Scissors

The computer will randomly choose its move
The result will be displayed:
You Win 🟢
You Lose 🔴
Draw ⚪

Score will update automatically

🔄 Additional Controls --------------
Reset Score → Reset all scores to 0
Auto Play → Let the game play automatically
