✊✋✌️ Rock Paper Scissors Game<br>

A simple and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript. Play against the computer and track your score in real-time!<br>

🚀 Features ------<br>
🎮 Play Rock, Paper, Scissors against the computer<br>
🤖 Random computer choice<br>
📊 Score tracking (Win, Lose, Draw)<br>
🔁 Reset score button<br>
⚡ Auto play mode<br>
🎨 Clean and modern UI design<br>

🛠️ Technologies Used ---------<br>
HTML – Structure of the game<br>
CSS – Styling and layout<br>
JavaScript – Game logic and interactivity<br>

🎯 How to Play -----------<br>
Click one of the buttons:<br>
✊ Rock<br>
✋ Paper<br>
✌️ Scissors<br>

The computer will randomly choose its move<br>
The result will be displayed:<br>
You Win 🟢<br>
You Lose 🔴<br>
Draw ⚪<br>

Score will update automatically<br>

🔄 Additional Controls --------------<br>
Reset Score → Reset all scores to 0<br>
Auto Play → Let the game play automatically<br>
